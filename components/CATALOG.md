# Component Catalog

> Generated by scripts/generate-catalog-md.sh — do not edit by hand.

A page in build-plan.yaml is a list of components stacked vertically.
Each component has a `type` field naming one of the entries below.

## gallery

Responsive image grid. Each image: { src (string), alt (string), caption (optional string) }.

**Required fields:**

- `images` (array)

**Optional fields:**

_(none)_

## mailto-form

Client-side contact form. On submit, composes a mailto: URL from field values and navigates to it. No backend.

**Required fields:**

- `to` (string)
- `fields` (array)

**Optional fields:**

- `subject` (string)
- `submit_label` (string)

## prose

Renders a Markdown body to HTML. Supports GFM: headings, paragraphs, lists, links, inline code, blockquotes, tables, fenced code blocks.

**Required fields:**

- `markdown` (string)

**Optional fields:**

_(none)_
